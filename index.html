Public Tarkov data can be synced from the tarkov.dev GraphQL API and cached locally. Local map images are loaded from this folder, not from another website.

About this local version

Local Tarkov Tracker (LTT) is an unofficial, free, local-first fan tool created by CtrlQuest. Updates and changes are published at https://github.com/CtrlQuest/Local-Tarkov-Tracker-LTT-.

The app code is released under the MIT License. Third-party game data, wiki content, API data and map images remain owned/licensed by their original creators. Offline map packs used in your local copy were sourced from the community Reddit map collection linked in the README.

How the raid-safe tracking works

  1. Press Track on an item you want to hunt.
  2. During raid, press Found in raid when you pick one up.
  3. If you extract, press I got out of the raid safely. The found items are added to your real progress.
  4. If you die, press I died / lost raid. The temporary raid bag is cleared and progress is not counted.

What was added from tarkov.dev-style info

  • Map page with extracts, transits, locks, bosses and switch notes when synced.
  • Key page with uses, map usage and room/lock references when synced.
  • Key Locker page for owned keys, needed keys and missing keys.
  • Mission/task page with collapsed objective cards, required items, FIR items, and detected key needs.
  • Story Chapters page with local objective checklists, item hints and key hints.
  • Hideout page with current level selection and next-upgrade materials.
  • Fallback offline seed data so the app is still useful before sync.
